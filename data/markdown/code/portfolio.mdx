## Description
The very site that you are on at the moment. It's a combined porfolio for my coding and music work.

It features a 3D model of me on the homepage and dedicated pages for both coding and music, each with
interactive 3D interfaces for navigating my projects.

## Technical Details
- **Site** built with [Next.js](https://nextjs.org/docs/app/guides/mdx), [react-three-fiber](https://github.com/pmndrs/react-three-fiber), and [GSAP](https://github.com/greensock/GSAP)
- **3D Ben** created with [photogrammetry](https://en.wikipedia.org/wiki/Photogrammetry) and refined in [Blender](https://www.blender.org/)
