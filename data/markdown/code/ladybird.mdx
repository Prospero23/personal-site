## Description
Ladybird is a truly independent web browser, using a novel engine based on web standards.


## Contribution
Implemented portions of the Web Audio API and contributed the [MP3-without-ID3](https://github.com/LadybirdBrowser/ladybird/pull/4807) and [WebM](https://github.com/LadybirdBrowser/ladybird/pull/4739) MIME sniffing algorithms.
This work also led to submitting a patch to the [MIME Sniffing Standard](https://mimesniff.spec.whatwg.org/).
